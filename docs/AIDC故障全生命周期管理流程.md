# AIDC 故障全生命周期管理流程（SOP V1.0）

> 覆盖故障发现、分级研判、应急响应、隔离定位、修复执行、恢复验证、复盘归档全流程。
>
> 撰写人：孟希東

---

故障分级：P0(灾难,5min) · P1(严重,15min) · P2(一般,30min) · P3(轻微,4h)

**Phase 1** 故障发现与告警确认：2min内确认 → 多源交叉验证 → 采集快照 → 创建工单

**Phase 2** 故障分级与影响研判：故障域判定 → 影响范围评估 → 定级 → 通知相关方

**Phase 3** 应急响应与任务保护：Checkpoint保护 → 任务迁移/暂停 → 扩散防护

**Phase 4** 故障隔离与根因定位：分域诊断 → 关联分析 → 根因评审

**Phase 5** 修复方案制定与执行：方案+审批+预检+执行+即时检查

**Phase 6** 恢复验证与业务回切：功能验证+压测≥1h+解除隔离+回切+观察期(P0/P1:72h)

**Phase 7** 复盘归档与持续改进：复盘会议+故障报告+知识库更新+改进跟踪+工单关闭(MTTD/MTTR)
