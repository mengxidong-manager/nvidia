# Dell 服务器 RAID 故障恢复操作手册

> 覆盖 RAID 0/1/5/10 四种级别的故障识别、硬盘更换、阵列重建完整流程。
>
> 撰写人：孟希東

---

RAID 0(无冗余) · RAID 1(镜像,坏1) · RAID 5(校验,坏1) · RAID 10(镜像+条带,每组坏1)

故障发现：指示灯(琥珀=故障) + iDRAC + `perccli /c0 show`

恢复流程：确认状态 → 热插拔更换 → 自动重建 → 确认 Optimal → 一致性检查

最佳实践：Hot Spare 热备盘 + 同型号替换盘 + 重建期降低 I/O + SMART 定检
