# 数据中心供电架构详解：双路市电、双柴发、UPS/电池、240V 直流

> 从电网到芯片的多级冗余保障链条——双路市电、双路柴油发电机、UPS/电池、240V HVDC 直流供电的架构与作用
>
> 撰写人：孟希東

---

## 一、整体架构层级

数据中心供配电系统是数据中心的"心脏和血管"，负责把能量输送到系统的每台设备，一旦发生故障会导致 IT 设备断电宕机，大型数据中心供电的任何故障都可能带来重大经济损失。

```
市政电网 (10-35kV AC)
    │
    ├── 路径A：市电 1 路 ──┐
    ├── 路径B：市电 2 路 ──┤
    │                      ▼
    │              高压变配电系统（变压器降至 380V/400V）
    │                      │
    ├── 备用：柴油发电机 1 ─┤ (通过 ATS 自动切换)
    ├── 备用：柴油发电机 2 ─┘
    │
    ▼
低压配电系统（母线/开关柜）
    │
    ▼
UPS 系统（不间断电源，含电池）── 或 HVDC 高压直流系统（240V DC）
    │
    ▼
列头柜 / 母线槽配电
    │
    ▼
机架 PDU（电源分配单元）
    │
    ▼
服务器 PSU（电源模块）
    │
    ▼
主板 / GPU（12V 或更低电压）
```

## 二、双路市电：第一道防线

### 作用

两路独立的市电引入（通常来自不同变电站或不同输电线路），互为备用，避免单一市电线路故障导致全站断电。

### 典型冗余架构

我国 A 级数据中心供配电系统主要有三种架构：

| 架构 | 说明 |
|------|------|
| **2N** | 由 2 个供配电单元组成，每个单元均能满足全部负载用电需要，2 个单元同时工作、互为备用；正常运行时每个单元提供 50% 电能，一个单元故障时另一个提供 100% |
| **DR（分布冗余）** | 由 N（N≥3）个供配电单元组成，形成"手拉手"供电方式，正常负荷率约 66%，一个单元故障时由相邻单元继续供电 |
| **RR（后备冗余）** | 多个供配电单元组成，其中一个作为其它单元的公共备用 |

高端数据中心通常采用 **2N 冗余系统**：从交流输入经 UPS 设备直到双电源输入负载，完全是两条彼此隔离的供电线路，供电路径中所有环节和设备都冗余配置。这种多路电源冗余方式能有效解决单电源系统的单点故障瓶颈，提高电力供应可靠性。

## 三、双柴油发电机：终极备用电源

### 作用

柴油发电机是数据中心电力保障的**最后一道防线**——当两路市电全部失去后，柴发是唯一能提供长时间（数小时到数天）电力支撑的电源。

### 关键参数与设计要点

| 参数 | 典型值 |
|------|--------|
| 启动延时 | 通常 ≤10 秒 |
| 燃料储备周期 | 标准为 72 小时或 168 小时（一周） |
| ATS 切换时间 | 毫秒级，通常 ≤10ms |

**ATS（Automatic Transfer Switch，自动转换开关）** 必须在毫秒级完成市电与发电机的无缝切换，且切换过程不得引发 IT 设备重启或存储阵列掉盘，这依赖于 ATS 的机械动作时间、同期检测精度、预同步合闸控制算法以及与 UPS 旁路的协同。

### 双柴发的意义

配置双路（2N）柴油发电机，是为了避免"柴发在启动时就出故障"这种灾难性场景——如果两路市电全部失去，而发电机在启动时发生停车保护，对数据中心来说就是灾难性事故。因此高等级数据中心的柴发系统本身也要做 2N 冗余，同时需要考虑变压器合闸瞬间冲击电流、UPS 启动时的电容特性、发电机功率圆图等匹配逻辑，避免发电机带载爬升时"带不动"。

### 与 UPS 的协同关系（时域接力）

**发电机-UPS-电池"三级能源缓冲"必须满足"时间接力"逻辑**：电池需要覆盖发电机冷启动期，发电机需要支撑至市电完全恢复或人工干预完成。

## 四、UPS + 电池：毫秒到分钟级的桥梁

### UPS 的核心作用

UPS 的主要作用，是在**市电电源中断、发电机启动之前**，确保所带的负载持续供电。UPS 系统包含储能设备（如蓄电池或飞轮），同时还具有隔离市电侧浪涌、电压骤升骤降等作用。

### UPS 的三大组成部分

传统在线式 UPS 主要由三部分电路组成：

1. **整流（AC-DC）**：市电正常时，先把交流整流为直流
2. **逆变（DC-AC）**：再把直流逆变为纯净交流供给负载，同时为电池浮充
3. **静态旁路**：正常情况下负载也可直接由旁路供电（ECO 模式），效率更高

市电中断时，蓄电池由浮充转放电，经逆变器为负载供电，对负载来说**感受不到输入端电源的中断**。

### 冗余架构演进

| 架构 | 特点 |
|------|------|
| N 系统 | 无冗余，成本低但可用性低，存在单点故障 |
| N+1 系统 | 增加一台备用 UPS，提升可用性 |
| 2N 冗余系统 | 两套完全独立的 UPS 系统，可靠性最高但成本也最高 |
| "市电+U电" | 百度提出的架构：双电源设备中一路改由市电直接供电，另一路走 UPS，消除单点故障的同时把 UPS 损耗降低约 50%，整体效率可提升至 95% 以上 |

### 时域保障体系

UPS 与柴油发电机组构成的复合供电体系，通过差异化技术特性形成多级电力保护屏障，未来将形成**毫秒-秒-分钟-小时**的四级保障机制：

| 保障层级 | 设备 | 响应时间 | 持续时长 |
|----------|------|----------|----------|
| 毫秒级 | 超级电容 | 极快 | 应对瞬时功率骤增（如 AI 算力指令突增） |
| 毫秒级 | BBU（电池备份单元） | 不超过 10ms | 最长约 4 分钟 |
| 毫秒级-分钟级 | UPS/HVDC | 不超过 10ms | 最长约 15 分钟 |
| 秒级-小时级 | 柴油发电机 | ≤10 秒 | 72-168 小时（视燃料储备） |

### BBU 与超级电容（AI 数据中心新增层级）

**BBU（电池备份单元）**：集成在机架中的含功率模块、锂电池的小容量备电设备，主要作用是当主电源中断或不足时，能够迅速接管供电，确保系统有足够的时间保存重要数据，并将操作转移到其他服务器。

**超级电容**：与 BBU 作用相似，还可以抑制电压波动，稳定 DC/DC 电源模块电压。主要用于服务器瞬时收到大量算力指令导致用电功率骤增的情况下提供瞬时电力支撑，持续时间较短。

四类备电方式（超级电容/BBU/UPS-HVDC/柴发）相互协同，共同构成数据中心完整的备电体系。

## 五、240V 直流（HVDC）：新一代供电架构

### 为什么会出现直流供电？

传统 UPS 架构中，交流市电需经过 **AC/DC 整流 + DC/AC 逆变**两次转换才能供给服务器，能量损耗较大。**HVDC（高压直流）架构省去了逆变环节**，只需一次 AC-DC 转换，效率更高。

### 国内 240V HVDC 标准

HVDC 由移动、联通等运营商推动并引入市场，**国内标准为 240V、336V，当前 240V 已成为国内主流标准，占比 90% 以上**。以百度、阿里、腾讯为主的互联网数据中心，通常采用 **市电+HVDC 双路供电架构**：一路 HVDC、一路市电直供，服务器双路电源一主一备同时热备份——市电直供损耗更低，HVDC 那一路保证市电中断时不间断供电。

### 完整链路对比

**传统 UPS 交流架构：**

```
市电 → 高压变配电 → 变压器降压(380V) → UPS(AC-DC-AC 两次转换) → PDU → 服务器 PSU(48V/12V DC)
```

**240V HVDC 直流架构：**

```
市电 → 高压变配电 → 变压器降压 → HVDC(仅一次 AC-DC 转换，输出 240V DC) → 直流配电 → 服务器 PSU
```

HVDC 相较 UPS 减少了逆变（DC-AC）环节，直接输出 240V/400V 直流电，转换效率更高，正逐渐成为智算中心的标配之一。

### HVDC 的核心组成

HVDC 核心组成为"换流站"，包括换流变压器、换流器、滤波器、平波电抗器、无功补偿装置等设备。与 UPS 一样，HVDC 也挂载蓄电池——市电中断时，同样利用蓄电池对负载持续供电。

### HVDC vs UPS 对比

| 维度 | 交流 UPS | 240V HVDC |
|------|----------|-----------|
| 转换环节 | AC-DC-AC（两次转换） | AC-DC（一次转换） |
| 理论效率 | 略低 | 理论更高，但实测两者均约 97% |
| 兼容性 | 通用型，兼容性好 | 服务器 PSU 需专门适配直流输入 |
| 维护安全性 | 交流有过零点，断路器操作相对安全 | 直流无法过零切换，断路器操作易产生拉弧，维护难度和安全隐患更高 |
| 占地空间 | 较大 | 更小 |
| 综合成本 | — | 设备成本略低，但加上直流断路器等配套后总体成本无明显优势 |

> 目前国内数据中心供电方案中，交流供电占比 80%-90%，仍是主流，但 AI 数据中心供电系统正呈现从交流 UPS 向 HVDC 发展的趋势。

### 服务器侧的直流利用

240V HVDC 输出后，经配电单元分配到机柜，服务器的 PSU（电源模块）将 240V DC 进一步转换为板级所需的 **48V 或 12V DC**，再经过主板上多级 DC-DC 转换（如 LLC 拓扑、多相 Buck 拓扑）最终降到 CPU/GPU 核心所需的约 0.7-1V。

## 六、演进趋势：巴拿马电源与 800V HVDC

**巴拿马电源**是阿里巴巴联合台达开发的方案，直接将 10kV 中压交流转换为 240V/336V 直流，把中压配电、移相变压器、整流模块、直流输出等环节集成在一套系统中，省去传统低压配电和冗余设备，大大降低占用体积，提升转换效率。

**电能路由器（EER）** 能够满足数据中心源网荷储一体化融合要求，有助于连接数据中心和绿色电力，未来有望逐步应用。

随着 GPU 机柜功率持续攀升——国内单机架功耗已从通算中心（传统数据中心）的 4-6kW 向智算中心（AIDC）的 20-40kW 迈进，未来有望逐步提升至 40-120kW 甚至更高——母线电压也在从 48V/240V 向 **400V/800V** 演进，以进一步降低大电流下的铜材用量和损耗：

- Meta、谷歌先行落地 **400V 侧置供电架构**，将电源、锂电 BBU、超级电容从 IT 机柜剥离至独立边柜，铜排直连服务器，相比 48V 电流降低 8.3 倍，铜材减少 30%，效率至 96.5%，适用于单机架 30-150kW 液冷智算集群
- 英伟达 2025 GTC 白皮书明确新一代 AI 基础设施采用 **800V 直流架构**，电网 10kV/400V AC 经 SiC 固态整流直接输出 800V DC，取消传统 UPS 逆变环节：同等功率下电流仅为 48V 的 1/16.7，铜材用量减少 45%，端到端供电效率提升 5% 以上；单台电源模块功率可突破 500kW，1MW 机柜仅需 2-3 个功率单元

这与此前 Vera Rubin / B300 NVL72 供电架构中提到的 800V HVDC 演进路线相互印证。
